# To-Do List App (Internship Project)

A modern, responsive, and feature-rich to-do list application built with vanilla HTML, CSS, and JavaScript (ES6+).

**Live Demo:** [https://buildsbysaif.github.io/interactive-todo-list-app/](https://buildsbysaif.github.io/interactive-todo-list-app/)

---

## Core Features

- **Full CRUD Functionality:** Add, edit, delete, and mark tasks as complete.
- **Persistent Local Storage:** All tasks and user preferences are saved in the browser, so data persists across sessions.
- **Graceful Validation:** A smooth shake animation provides feedback for empty task submissions.
- **Responsive Design:** The application is fully responsive for desktop, tablet, and mobile devices.

## Advanced Features Implemented

- **Dark/Light Mode:** A user-friendly theme switcher with the user's preference saved for future visits.
- **Task Filtering:** Users can filter tasks by "All," "Active," and "Completed" states.
- **Professional UI/UX:** A clean interface featuring a custom logo, tagline, professional typography, and smooth transitions.
- **Modular Code Architecture:** The JavaScript is structured into ES6 Modules for maintainability and scalability.

---

## How to Run Locally

1.  Clone the repository: `git clone https://github.com/buildsbysaif/interactive-todo-list-app.git`
2.  Navigate to the project directory.
3.  Open the `index.html` file in any modern web browser.

## Technologies Used

- HTML5
- CSS3 (Flexbox, CSS Variables, Animations)
- Vanilla JavaScript (ES6 Modules)
